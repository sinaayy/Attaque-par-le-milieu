Afin de pouvoir executer le code, vous aures besoin d'une version récente de python (python 3.10.X).
(lien vers la page de téléchargement de python : https://www.python.org/downloads/)
Puis pour executer le programme vous devrez taper dans le terminal (dans le répertoire du dossier)
python attaque_milieu.py (python3 attaque_milieu.py sur certaines machines)
Le programme vous demandera d'entrer ensuite les couples m1 c1 m2 c2 puis le proramme effectura une 
attaque par le milieu. Selon les machines, le temps d'execution peut grandement varier mais il devrait
être compris entre 30 min et 1H30. Une fois l'attaque terminé, vous verrez affiché dans le terminal
les couples de clé k1k2 qui ont pu etre utilisé pour chiffre votre m1 et votre m2.
(les couples de clé valides k1k2 seront également écrit dans le fichier cle_valides.txt)
